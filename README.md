# Online Library Management System (C++)

## Overview
This project implements an Online Library Management System using Object-Oriented Programming (OOP) principles in C++.  
It manages books, users, and borrowing/returning operations.

## Features
- Add/remove books
- Search by ISBN
- Register users
- Borrow and return books

## Installation & Execution
```bash
g++ main.cpp Book.cpp User.cpp Library.cpp -o library
./library
